## Цель

Лендинг о даосском мастере Чжан Чжи Шунь (张至顺) и его практиках цигун. Сайт представляет мастера, его биографию и два комплекса упражнений (金刚功 и 长寿功) по 8 упражнений каждый.

---

## Архитектура

**Тип:** Single-page application (один длинный скролл, без React Router).  
**Причина:** контент пока в заглушках, роутинг добавить легко позже.  
**i18n:** `react-i18next` с тремя локалями — `ru` (приоритет), `en`, `zh`. Все строки в файлах переводов с первого дня.

## Секции (сверху вниз)

### 1. Navbar (fixed)
- Логотип: иероглифы «炁體源流» золотом слева
- Справа: переключатель RU / EN / ZH
- Фон: `linear-gradient(180deg, #060c18ee, transparent)` — не перекрывает hero
- При скролле: добавляется `backdrop-blur` и полупрозрачный фон

### 2. Hero (100vh)

**Слои (z-index снизу вверх):**
1. Звёздное поле — редкие точки через CSS `radial-gradient`, без анимации на старте (z:0)
2. Горы — CSS `clip-path`, занимают нижние 45% экрана (z:1)
3. Фигура мастера — PNG с прозрачным фоном по центру, высота ~70vh (z:2). Ноги мастера перекрываются горами (z:1) для эффекта глубины
4. Туман — `linear-gradient` у нижнего края, поверх ног мастера (z:3)
5. Текст сверху: подзаголовок + иероглифы «炁體源流» крупно золотом (z:4)
6. Имя мастера снизу: «张至顺» + «Чжан Чжи Шунь» (z:4)
7. Анимированная подсказка «↓ прокрути» (fade-pulse, z:4)

**Анимация входа (Framer Motion):**
- Иероглифы — `opacity 0→1`, `y: -20→0`, задержка 0.3s
- Фигура мастера — `opacity 0→1`, `scale: 0.97→1`, задержка 0.6s
- Горы — `opacity 0→0.8`, задержка 0.9s

### 3. Биография (свиток)

- Центрированный блок, ширина max 500px
- Сверху и снизу — деревянный стержень (CSS gradient, gold + brown)
- Тело свитка: на десктопе раскрывается по ховеру, на мобайле — по клику (toggle)
- Анимация: Framer Motion `AnimatePresence` + `height: "auto"`
- Внутри: заголовок «张至顺 · Биография мастера» + текст на бумажном фоне
- Подсказка под свитком: «— наведи курсор, чтобы раскрыть свиток —» (скрывается после первого открытия)

### 4. 金刚功 — Алмазная крепость

- Заголовок секции: иероглифы крупно (золото) + русское название мелко
- Разделитель: тонкая линия `linear-gradient(transparent → gold → transparent)`
- Сетка: `grid-template-columns: repeat(4, 1fr)`, gap 20px, max-width 480px
- На мобайле: `repeat(2, 1fr)`

**Каждый портал:**
- Круг 100×100px (на десктопе), 80×80px (мобайл)
- Обводка: `2px solid #4488cc66`, при ховере яркость увеличивается
- Фон: `radial-gradient(circle at 40% 35%, #1a4080, #060e1e)`
- Свечение: `box-shadow: 0 0 15px #4488cc33, inset 0 0 20px #0a2040`
- Туман: `linear-gradient` у нижнего края круга
- Заглушка внутри: иконка + текст «упр. N»
- Ярлык под кружком: «N · Название» в маленьком капсуле синего цвета
- При клике: открывается модальное окно

### 5. 长寿功 — Практика долголетия

Идентичная структура с секцией 4. Другой оттенок фона (`#050a14`) для визуального разделения.

### 6. Footer

- Иероглифы «炁體源流» золотом
- Подпись: «张至顺 · Школа даосских практик»
- Разделитель сверху: `1px solid #d4a85322`

## Модальное окно упражнения

Открывается при клике на любой портал. Закрывается: клик на фон, кнопка «✕», клавиша `Escape`.

**Структура:**
```
┌─────────────────────────────────┐
│  [金刚功 · Упражнение 1]        │  ← тег секции
│  ╔═══════════════════════════╗  │
│  ║   [видео/фото заглушка]   ║  │  ← 200px, синий фон + кнопка ▶
│  ╚═══════════════════════════╝  │
│  Подъём Неба                    │  ← h3 золотом
│  Текст описания упражнения...   │  ← muted, line-height 1.7
│  ─────────────────────────────  │
│  [← Пред.]  [✕ закрыть]  [След. →] │
└─────────────────────────────────┘
```

**Анимация (Framer Motion):**
- Backdrop: `opacity 0→1`
- Карточка: `opacity 0→1`, `scale: 0.95→1`, `y: 20→0`

```

  locales/
    ru/translation.json   ← приоритет
    en/translation.json   ← заглушки
```
